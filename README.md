## BXSPHP（百晓生PHP简易框架）
程序配置方式和ThinkPHP相同 <br/>
将网站访问目录设置为public <br/>
配置好config.php内容后，将sql文件导入数据库 <br/>

框架集成了一些我自己常用的模块，如果不需要的可以删除掉 <br/>
