## BXSPHP 1.0

#### 介绍
百晓生PHP简易框架，简易，高效，便捷

#### 软件架构
框架参考了ThinkPHP的工作模式开发，但更方便，更适合小型开发


#### 安装教程

1.  下载代码文件释放到网站根目录
2.  编辑config.php内容并保存
3.  将sql数据文件导入数据库(模块支持)
4.  设置网站访问目录为public
5.  Nginx按照说明设置号伪静态规则
